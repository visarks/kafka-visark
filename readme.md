> 本软件使用`javafx`开发的 kafka 跨平台客户端

# 官网
https://www.visark.cn/

# 功能列表
1. topic的创建,删除,查看
2. partition的新增
3. 消息的发送,消费,查询(基于 时间,offset,partition)
4. consumer的消费情况
